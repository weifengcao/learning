# cloud

This folder is for the infrastructure and distributed-systems foundation needed to run modern applications.

Cover:

- compute, storage, and networking primitives
- containers and images
- Kubernetes and orchestration basics
- service discovery and load balancing
- distributed systems basics
- reliability, redundancy, and failure domains
- observability infrastructure
- deployment and release patterns

This folder should also carry networking and distributed-systems study for now. If either topic grows large enough, promote it to a dedicated top-level folder later.
