# saas

This folder is for traditional SaaS architecture and product foundations.

Cover:

- multitenancy
- RBAC and user management
- billing and metering
- configuration and feature flags
- audit logs
- workflow modeling
- APIs and integrations
- control plane and data plane concepts

This folder matters because enterprise AI products still inherit many SaaS constraints. AI changes the interaction model, but not the need for strong tenancy, auditability, and operational discipline.
