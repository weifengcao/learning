# llm

This folder is for model-side LLM foundations.

Cover:

- transformer basics
- tokenization
- pretraining and fine-tuning concepts
- inference and decoding
- embeddings
- reasoning limitations
- evaluation methodology
- model capability and cost tradeoffs

Relationship to `enterprise_ai_app/`:
Use this folder to understand the underlying model technology. Use `enterprise_ai_app/01_llm_foundations/` for application design patterns built on top of those models.
