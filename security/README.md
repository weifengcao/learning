# security

This folder is for security fundamentals that apply across software systems and enterprise AI products.

Cover:

- authentication and session management
- authorization models
- secrets management
- secure coding practices
- threat modeling
- common web vulnerabilities
- cloud security basics
- supply-chain and dependency risk
- encryption basics

Security should stay a standalone foundation. Enterprise trust in AI systems depends on strong access control, safe execution boundaries, and good auditability.
