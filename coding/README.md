# coding

This folder is for programming fluency and software engineering fundamentals.

The immediate focus here is Python skill growth across two complementary tracks:

- `ds/algo`: improve problem solving, data structure fluency, and implementation precision
- `AI engineering`: improve practical Python skills for LLM apps, data pipelines, evaluation, and tool-driven workflows

Cover:

- language fundamentals
- data structures and algorithms
- clean code and refactoring
- testing strategy
- debugging and profiling
- API design
- concurrency basics
- package and dependency management
- code review discipline

Recommended structure:

- `python/`
- `PYTHON_LEARNING_PLAN.md`

## Working rule

Use this folder for implementation practice.

- if the goal is writing better Python, the work belongs here
- if the goal is understanding model theory, it belongs in `llm/`
- if the goal is enterprise AI system architecture, it belongs in `enterprise_ai_app/`

## Main outcomes

By working through this folder, you should be able to:

- write clearer and more reliable Python
- solve `ds/algo` problems with stronger rigor and speed
- build AI-oriented Python utilities and services without hand-wavy code
- test, debug, and profile Python code in a disciplined way
