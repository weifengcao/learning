# 00_foundations

Purpose:
Build Python fluency before trying to optimize for speed or advanced AI workflows.

Cover:

- syntax and control flow
- functions and modules
- typing
- exceptions
- file handling
- JSON and CSV processing
- `pytest` basics
- debugging with logs and interactive inspection

Suggested exercises:

- parse a CSV file into typed objects
- build a simple CLI task runner
- write unit tests for a small utility module
- refactor a script into a package with clear functions

Exit criteria:

- you can write small Python programs without constant syntax lookup
- you can structure code into modules and tests
- you can read stack traces and debug basic failures
