# python

Use this folder to build strong implementation habits in Python.

This Python track is split into two main skill directions:

- `01_ds_algo/` for implementation precision and complexity thinking
- `02_ai_engineering/` for practical Python used in AI workflows and services

Those tracks share the same foundation in `00_foundations/` and converge in `03_projects/`.

If you are preparing for interviews, use `01_ds_algo/interview_practice/` for generic and company-style Python practice.

Cover:

- syntax and standard library fluency
- typing
- data modeling
- async programming
- testing with fixtures and mocks
- packaging
- performance basics
- common backend patterns

Python is a practical anchor language for AI systems, workflow backends, data pipelines, and integration-heavy enterprise tools.

## Recommended progression

1. Build fluency in `00_foundations/`
2. Practice implementation rigor in `01_ds_algo/`
3. Build AI-oriented components in `02_ai_engineering/`
4. Combine both in `03_projects/`
