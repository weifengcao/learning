## Tier 1: Core interview reps

These are the best warm-up problems for building speed and correctness in Python.

1. Reverse a linked list | linked list pointers
2. Determine sanitized palindrome in string | string cleanup and two pointers
3. Check perfect square using binary search | binary search
4. Detect intersection of two linked lists | two pointers
5. Find the k-th largest element in an array | heap or quickselect
6. Count connected land components in a grid | BFS or DFS on a grid
7. Find lowest common ancestor | tree recursion
8. Analyze time and space complexity | explanation discipline

## Tier 2: Core pattern depth

These build breadth across graph, tree, DP, and sliding-window patterns.

1. Compute inverse-depth weighted sum of nested lists | nested traversal
2. Group words that map to same phone digits | encoding and hashing
3. Find dictionary words matching a phone digit string | mapping and backtracking
4. Solve common string and subarray problems | mixed string and array patterns
5. Count islands and distinct shapes | DFS plus shape normalization
6. Find k closest values in a BST | BST traversal
7. Minimize adjacent-color assignment cost | dynamic programming
8. Solve min window & animal conflicts | sliding window
9. Solve substring and top-K pair problems | sliding window plus heap
10. Solve six algorithmic problems | mixed interview reps

6. Merge two N-ary trees by key rules | recursive tree merge with maps
