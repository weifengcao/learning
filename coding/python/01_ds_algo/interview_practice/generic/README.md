# generic

Purpose:
Cover the core Python interview patterns that appear across many companies.

## Starter set

Arrays and strings:

- Two Sum
- Best Time to Buy and Sell Stock
- Product of Array Except Self
- Longest Substring Without Repeating Characters

Hash maps and grouping:

- Valid Anagram
- Group Anagrams
- Top K Frequent Elements

Intervals and searching:

- Merge Intervals
- Insert Interval
- Search in Rotated Sorted Array

Trees and graphs:

- Binary Tree Level Order Traversal
- Lowest Common Ancestor of a Binary Tree
- Number of Islands
- Course Schedule

Backtracking and dynamic programming:

- Combination Sum
- Coin Change
- House Robber

Data structure design:

- Min Stack
- LRU Cache

## Why this set matters

This group covers the patterns that most directly improve Python interview performance:

- dictionary and set fluency
- indexing and pointer manipulation
- recursion and traversal
- complexity analysis
- state management in mutable structures

## Practice order

1. arrays and strings
2. hash maps and sets
3. intervals and binary search
4. trees and graphs
5. backtracking and dynamic programming
6. data structure design
