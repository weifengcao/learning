# agent

This folder is for agent-system foundations beyond narrow enterprise deployment concerns.

Cover:

- reasoning loops and control flow
- planning strategies
- tool-use protocols
- short-term and long-term memory patterns
- environment interaction and state tracking
- single-agent vs multi-agent design
- failure modes such as looping, drift, and tool misuse

Relationship to `enterprise_ai_app/`:
Use this folder for general agent techniques. Use `enterprise_ai_app/03_workflows_and_agents/` for enterprise-safe workflow design and production controls.
